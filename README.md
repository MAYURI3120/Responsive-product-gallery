Responsive Product Gallery

This is a training project to create a responsive product gallery using HTML, CSS, and JavaScript.

Project Overview:
- 6 product cards created dynamically using JavaScript
- Each card shows:
  - Image
  - Title
  - Price
  - "View Details" button
- Clicking "View Details" opens a modal popup
  - Shows bigger image and description
  - Can close with X button or clicking outside

Features:
- Responsive layout:
  - Desktop: 4 cards per row
  - Tablet: 2 cards per row
  - Mobile: 1 card per row
- Clean blue theme (light & dark blue)
- Simple and student-level design

Technologies Used:
- HTML
- CSS (Grid + Media Queries)
- JavaScript (DOM Manipulation & Events)

How to Run:
1. Open the folder
2. Open index.html in any browser

Screenshots:
- Desktop view: Desktop_UI.png
- Mobile view: Mobile_UI.png, Mobile_UI_m.png

Notes:
- Focused on layout, responsiveness, and JS DOM functionality
- Designed to be simple, clean, and easy to understand
